# Bionic Arm

## Project Description

The Bionic Arm is an EMG-controlled prosthetic hand designed to provide an affordable and intuitive assistive solution for individuals with wrist-level hand amputations. The primary objective of this project is to bridge the gap between high-cost commercial prosthetics and accessible assistive technology by utilizing low-cost hardware, embedded systems, and biomedical signal processing.

The system operates by detecting electrical signals generated by muscle contractions in the user's forearm. These signals, known as Electromyography (EMG) signals, are captured using EMG sensors placed on specific muscle groups. When a user intentionally contracts a muscle, the sensor measures the resulting electrical activity and sends it to an ESP32 microcontroller for processing.

The ESP32 analyzes the incoming EMG data, filters out unwanted noise, and identifies specific muscle activation patterns. Based on predefined thresholds and control logic, the system determines the intended hand movement and generates appropriate control signals for the actuators. Servo motors integrated into the prosthetic hand then convert these commands into physical finger movements, allowing the user to perform actions such as opening, closing, and gripping objects.

The project focuses on creating a lightweight, modular, and scalable design that can be further improved through advanced gesture recognition algorithms and machine learning techniques. By combining embedded electronics, signal processing, and mechanical design, the Bionic Arm aims to provide a practical platform for prosthetics research and assistive technology development.
